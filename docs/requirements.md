# 拼豆图案设计器 — 需求文档

## 项目简介

面向拼豆爱好者的像素图案设计工具。支持网格绘制、图片转拼豆图案、多板拼接规划，后续扩展图纸广场社区功能。

## 技术栈

- React 18 + TypeScript
- Vite 构建
- HTML5 Canvas 编辑器
- Zustand 状态管理
- Dexie.js（IndexedDB 封装）
- 霓虹像素风 UI

## 品牌色板

内置 Artkal S 系列（2.6mm）色板数据，后续扩展 Hama 和 Perler。

## 小程序嵌入

编辑器页面通过微信小程序 `<web-view>` 嵌入，用户系统和广场用小原生开发。

## 版本规划

| 版本 | 内容 |
|---|---|
| MVP | 画板编辑器 + 图片转拼豆 + PNG导出 + 本地保存 |
| V2 | 多板拼接 + 用料统计 + 社区广场 |
